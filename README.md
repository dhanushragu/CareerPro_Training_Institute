## Overview

**Career Pro** is a modern, responsive, single-page Career Training Institute website developed using **HTML5, CSS3, and Vanilla JavaScript (ES6)**. The website is designed to provide a premium user experience while showcasing the institute's courses, trainers, placement support, and contact information.

The project is fully front-end based and does not require any backend technologies, making it suitable for deployment on **GitHub Pages** and inclusion in a professional web development portfolio.

---

## Features

* Modern and professional user interface
* Fully responsive design for Mobile, Tablet, Laptop, and Desktop
* Sticky navigation bar
* Smooth scrolling navigation
* Active navigation highlighting
* Animated hero section
* Professional About section
* Course cards with hover effects
* Why Choose Us section
* Animated placement statistics counters
* Professional trainer profiles
* Automatic testimonial slider
* FAQ accordion
* Contact form with JavaScript validation
* Embedded Google Maps
* Back-to-top button
* Loading screen
* Scroll reveal animations
* Clean, semantic HTML structure
* Optimized for GitHub Pages

## Installation

1. Download or clone this repository.
2. Extract the project files if downloaded as a ZIP.
3. Add all required images to the **images** folder.
4. Open **index.html** in any modern web browser.

No installation or package manager is required.

---

## Deployment on GitHub Pages

1. Create a new repository on GitHub.
2. Upload all project files while maintaining the original folder structure.
3. Commit and push the files.
4. Open the repository **Settings**.
5. Navigate to **Pages**.
6. Under **Source**, select:

   * **Deploy from a branch**
   * **Branch:** main
   * **Folder:** / (root)
7. Save the settings.
8. GitHub will generate a public website URL.
